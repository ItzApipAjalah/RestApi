Twitter/X Downloader

Overview

Download videos and images from Twitter/X posts.

Endpoint

POST /api/download/twitter

Request Body

{
    "url": "https://twitter.com/username/status/1234567890"
}

Response

{
    "success": true,
    "message": "Media found",
    "data": {
        "type": "video|image",
        "url": "https://example.com/media.mp4"
    }
}

Example Usage

fetch('/api/download/twitter', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        url: 'https://twitter.com/username/status/1234567890'
    })
})
.then(response => response.json())
.then(data => console.log(data));
