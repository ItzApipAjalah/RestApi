Anime Image Scraper

Overview

Scrape anime images from a given code.

Endpoint

POST /api/scrape/anime

Request Body

{
    "code": "123456"
}

Response

{
    "success": true,
    "message": "Images found",
    "data": {
        "code": "123456",
        "totalPages": 24,
        "images": [
            {
                "page": 1,
                "url": "https://example.com/image1.jpg"
            },
            {
                "page": 2,
                "url": "https://example.com/image2.jpg"
            }
            // ... more images
        ]
    }
}

Example Usage

fetch('/api/scrape/anime', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        code: '123456'
    })
})
.then(response => response.json())
.then(data => console.log(data));
